# Smart Home Electricity Monitoring & Control System

## Project Overview

The **Smart Home Electricity Monitoring & Control System** is an ESP32-based IoT solution designed to monitor and control household electrical appliances remotely. The system measures electrical parameters such as voltage, current, power consumption, and energy usage in real time while providing remote appliance control through Arduino IoT Cloud.

---

## Features

- Real-time voltage monitoring
- Real-time current monitoring
- Power consumption calculation
- Energy unit calculation (kWh)
- Electricity cost estimation
- Remote appliance control via Arduino IoT Cloud
- Live data display on 16x2 LCD
- EEPROM-based data storage
- Wi-Fi connectivity for cloud integration

---

## Components Used

- ESP32 Development Board
- Voltage Sensor Module
- Current Sensor Module
- 16x2 LCD Display
- 2-Channel Relay Module
- Wi-Fi Network
- Arduino IoT Cloud Platform

---

## System Architecture

1. Voltage and current sensors continuously monitor electrical parameters.
2. The ESP32 processes sensor data and calculates:
   - Voltage
   - Current
   - Power
   - Energy Consumption
   - Estimated Electricity Cost
3. The calculated values are displayed on the LCD screen.
4. Data is uploaded to Arduino IoT Cloud through Wi-Fi.
5. Users can monitor energy usage and control connected appliances remotely using the cloud dashboard.
